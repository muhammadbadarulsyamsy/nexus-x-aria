# No-Go Platforms v0.1

This is not a permanent blacklist. It is a “do not use yet” list.

## Avoid for Now

### RapidAPI

Reason:

- marketplace orientation
- monetization pressure
- pricing/payout setup risk
- abuse control not ready
- API key/rate-limit model not ready

### Paid API Marketplaces

Reason:

- ARIA is still MVP
- no paid support process
- no abuse control
- no custom rate limit
- no pricing model

### Payout-Based Platforms

Reason:

- may require personal/payment setup
- too early for monetization
- may distract from product hardening

### Platforms Requiring Billing/Card

Reason:

- project currently follows zero/low-cost-first workflow
- user asked to stop if billing/card appears

### Spam-Oriented Traffic Platforms

Reason:

- harms credibility
- may violate rules
- not passive publication

### Communities That Ban Promotion

Reason:

- respect community rules
- avoid spam
- use educational article only where appropriate

### Platforms That Prohibit AI-Assisted Content Without Disclosure

Reason:

- must follow platform rules
- if disclosure is required, disclose properly or do not post

## Stop Immediately If Asked For

- billing
- card
- upgrade
- payout
- API token
- secret
- private key
- seed phrase
- password
- OTP
- cookie
- wallet connection
