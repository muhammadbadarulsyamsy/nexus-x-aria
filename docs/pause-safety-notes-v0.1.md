# Pause Safety Notes v0.1

## Purpose

Document what must not happen during a maintenance pause.

## Do Not Start

```text
runtime v0.2.1 without bug evidence
API key system
login system
database
external reputation feed
blockchain explorer integration
wallet connection
transaction simulation
monetization
RapidAPI
paid marketplace
Postman public publishing
```

## Stop Immediately If Asked For

```text
billing
card
upgrade
payout setup
API token
secret
password
OTP
cookie
private key
seed phrase
wallet connection
```

## Safe During Pause

```text
docs review
repo review
README alignment
manual checklist updates
parking lot review
memory database append-only updates after completed phases
```

## Public Claim Guard

Do not claim:

```text
guaranteed scam detection
wallet protection
financial advice
complete Web3 security product
```

Use:

```text
heuristic risk analysis
educational/informational tool
may produce false positives or false negatives
always verify independently
```
