# Public Value Message Polish v0.1

## Purpose

This document defines the public-facing value message for NEXUS-X ARIA.

It is intended to make ARIA understandable to external users in 30 seconds.

This is a docs-only positioning milestone.

## Recommended Headline

```text
Transparent heuristic risk scoring for Web3 airdrop metadata.
```

## Recommended Short Description

```text
NEXUS-X ARIA is a lightweight serverless API that helps developers inspect common Web3 airdrop risk signals with explainable heuristic scoring.
```

## One-Sentence Description

```text
NEXUS-X ARIA analyzes airdrop project metadata and returns a risk score, risk level, red flags, safe actions, and explainable rule-based scoring details.
```

## 30-Second Pitch

```text
NEXUS-X ARIA is a lightweight serverless API for reviewing Web3 airdrop metadata. It uses transparent heuristic rules to return a risk score, red flags, safe actions, and explainable scoring details such as rule_hits and score_breakdown. It is built for developers and educators who want a simple way to inspect common airdrop risk signals without connecting wallets or handling sensitive credentials.
```

## Indonesian 30-Second Pitch

```text
NEXUS-X ARIA adalah API serverless ringan untuk meninjau metadata airdrop/Web3. ARIA memakai rule heuristik transparan untuk menghasilkan skor risiko, red flag, tindakan aman, dan detail scoring seperti rule_hits dan score_breakdown. ARIA dibuat untuk developer dan edukator yang ingin membaca sinyal risiko umum tanpa connect wallet dan tanpa menangani kredensial sensitif.
```
