# Manual Publication Package v0.1

## Purpose

This package prepares one safe public publication package for NEXUS-X ARIA.

It is based on the Manual Publication Test Plan and Public Value Message Polish.

This is docs-only.

## What This Package Provides

```text
one safe article/post draft
one short public post copy
one pre-publish link checklist
one publish checklist
one safety review checklist
```

## Current Position

NEXUS-X ARIA is technically ready as a v0.2.0 MVP.

It has:

```text
public Cloudflare Worker API
OpenAPI spec
Try page
Examples page
Postman local collection
25/25 test pack
public value message
manual publication test plan
```

## Goal

The goal is not monetization yet.

The goal is to publish one safe, clear, non-spam public post and observe whether there is any interest signal.

## Target Audience

```text
Web3 developers
educators
open-source builders
airdrop dashboard builders
security education writers
```

## Primary Message

```text
NEXUS-X ARIA is a lightweight serverless API that helps developers inspect common Web3 airdrop risk signals with explainable heuristic scoring.
```

## Publication Rule

Post manually only.

Do not automate.

Do not spam.

Do not mass-DM.

Do not claim guaranteed scam detection.

Do not claim wallet protection.

Do not ask for secrets.
