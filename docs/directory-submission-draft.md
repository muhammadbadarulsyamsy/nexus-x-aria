Long description:

```text
NEXUS-X ARIA is a transparent heuristic airdrop risk scoring API for Web3 projects. It accepts metadata such as project URL, description, required tasks, chain, token contract, and social links, then returns a risk score, risk level, verdict, red flags, safe actions, and summary.

It includes a browser Try Page, Examples page, documentation, OpenAPI 3.0.1 specification, and internal test pack. ARIA is designed for educational and informational use and does not guarantee safety.
```
