# Phase 8.3 — v0.2.1 Ideas Parking Lot v0.1

## Purpose

This document parks possible v0.2.1 ideas without starting runtime development.

Phase 8.3 is docs-only.

It prevents premature coding after the stable v0.2.0 release.

## Current Stable Baseline

```text
Runtime: v0.2.0
Release: v0.2.0
Test pack: 25 tests, 25 passed, 0 failed
```

## Why Parking Lot Exists

NEXUS-X ARIA v0.2.0 is stable and released.

Before changing runtime again, every idea must be parked, described, evaluated, and accepted through a release gate.

## Parking Lot Rules

```text
[ ] Ideas are not commitments.
[ ] Ideas do not automatically become v0.2.1.
[ ] Runtime changes require clear bugfix or scoped improvement.
[ ] No monetization-driven feature jumps.
[ ] No API key system unless separately approved later.
[ ] No database unless separately approved later.
[ ] No external reputation feed unless separately approved later.
```

Use this format for each idea:
