# Listing Copy — Short

## English Version

NEXUS-X ARIA is a transparent heuristic airdrop risk scoring API for Web3 projects. It helps identify common risk signals and returns a risk score, risk level, verdict, red flags, safe actions, and summary. Educational use only; not a guarantee of safety.
