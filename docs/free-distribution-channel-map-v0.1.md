# Free Distribution Channel Map v0.1

## Purpose

Map possible free distribution channels for NEXUS-X ARIA.

This is a planning document, not a command to publish everywhere.

## Channel Evaluation Rules

Each channel must be checked for:

```text
free to use
manual posting allowed
no billing required
no card required
no credential sharing
no spam behavior
no misleading claims
```

## Candidate Channels

### 1. GitHub Repository

Use case:

```text
primary project home
README clarity
release links
OpenAPI/Postman/docs discovery
```

Signal to watch:

```text
stars
forks
issues
watchers
profile views if available
```

Risk:

```text
low
```

### 2. Dev.to Manual Article

Use case:

```text
technical story
open-source build log
developer audience
```

Signal to watch:

```text
views
reactions
comments
profile follows
clicks if visible
```

Risk:

```text
low to medium
```

Stop if:

```text
platform asks for payment
platform asks for suspicious credential actions
```

### 3. Existing Social/Profile Post

Use case:

```text
short announcement
link to GitHub/demo
```

Signal to watch:

```text
replies
likes
clicks if visible
stars after posting
```

Risk:

```text
medium if spammy
```

Rule:

```text
one clean post only
no mass DM
no repeated promotion
```

### 4. Developer Communities

Use case:

```text
feedback from developers/builders
```

Signal to watch:

```text
comments
questions
criticism
feature requests
```

Risk:

```text
medium
```

Rule:

```text
only post where project sharing is allowed
do not disguise promotion as help
do not spam
```

## Do Not Use Yet

```text
paid API marketplaces
RapidAPI monetization
payout setup
ad campaigns
cold DM
wallet-based communities requiring connect wallet
platforms requiring card/billing
```
