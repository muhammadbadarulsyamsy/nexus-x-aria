# v0.2.1 No-Go Ideas v0.1

## Purpose

This document lists ideas that are not allowed for v0.2.1 without a separate governance decision.

## No-Go for v0.2.1

```text
API key system
login system
user accounts
database
external reputation feed
blockchain explorer integration
wallet connection
transaction simulation
paid marketplace launch
RapidAPI monetization
Postman public publishing
payout setup
billing setup
card setup
```

## Why These Are No-Go

These ideas increase complexity, cost, or security risk.

They also conflict with the current stability-first phase after v0.2.0.

## Stop Conditions

Stop immediately if any next step asks for:

```text
billing
card
upgrade
payout setup
API token
secret
password
OTP
cookie
private key
seed phrase
wallet connection
```

## Allowed Instead

```text
docs-only planning
manual checklist
regression checklist
false-positive review
README alignment
idea parking
```
