# Allowed Claims and Forbidden Claims

## Safe Claims

These claims are acceptable:

- NEXUS-X ARIA provides heuristic airdrop risk scoring.
- ARIA is educational and informational.
- ARIA uses transparent rule-based scoring.
- ARIA returns risk score, risk level, verdict, red flags, safe actions, and summary.
- ARIA includes a browser Try Page.
- ARIA includes an examples page.
- ARIA provides an OpenAPI specification.
- ARIA includes an internal test pack.
- ARIA does not connect to wallets.
- ARIA does not request private keys.
- ARIA does not request seed phrases.
- ARIA does not store user funds.
- ARIA is a serverless Cloudflare Worker MVP.

## Claims That Need Care

Use carefully and with context:

- phishing detection
- scam detection
- risk intelligence
- security tool
- wallet safety

Prefer:

> helps identify common risk signals

Instead of:

> detects scams

## Forbidden Claims

Do not claim:

- guaranteed safety
- detects all scams
- prevents phishing
- prevents wallet theft
- certified security audit
- financial advice
- investment advice
- legal advice
- enterprise-grade security
- production-ready fraud prevention
- guaranteed accurate scoring
- zero false positives
- zero false negatives

## Required Disclaimer

Use this disclaimer or similar:

> NEXUS-X ARIA provides heuristic analysis for educational and informational purposes only. It does not guarantee safety and is not financial advice. Users must verify independently and must never share private keys, seed phrases, recovery phrases, passwords, OTPs, cookies, API keys, or wallet credentials.
