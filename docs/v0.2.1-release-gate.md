# v0.2.1 Release Gate

## Allowed Reasons for v0.2.1

```text
documented bugfix
documented false positive
documented false negative
documentation/runtime version mismatch
minor compatibility fix
small test-pack improvement
```

## Not Allowed Reasons

```text
monetization
RapidAPI
Postman public publishing
paid marketplace
API key system
database
external reputation feed
major refactor
feature curiosity
```

## Required Before v0.2.1 Runtime Patch

```text
[ ] clear issue or improvement reason
[ ] expected behavior defined
[ ] test case added or updated
[ ] no credential needed
[ ] no paid service needed
[ ] no database needed
[ ] no external API needed
```

If there is no urgent bug, stay on v0.2.0 and focus on stability.
