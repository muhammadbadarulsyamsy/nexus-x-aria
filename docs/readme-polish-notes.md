## Why This Matters

The README is the first thing most GitHub visitors see. It should explain:

- what ARIA is,
- what it does,
- where to try it,
- what its limitations are,
- what claims are safe,
- what users must never do.

## Safety Position

ARIA is positioned as:

```text
a transparent heuristic educational MVP
```

ARIA is not positioned as:

```text
a guaranteed scam detector
a financial advisor
a certified security auditor
an enterprise security product
```
