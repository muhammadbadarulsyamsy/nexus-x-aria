# Repo Public Safety Review v0.1

## Purpose

Ensure public-facing repository material does not overclaim NEXUS-X ARIA capabilities.

## Required Safety Position

NEXUS-X ARIA must remain described as:

```text
heuristic-only
educational
informational
not financial advice
not a security guarantee
not a complete security product
not a wallet tool
```

## Forbidden Claims

Do not claim:

```text
guaranteed scam detection
protects your wallet
prevents all phishing
financial advice
complete Web3 security product
approved by a chain or wallet provider unless verified
```

## Forbidden Inputs

Examples and docs must never request real:

```text
private keys
seed phrases
recovery phrases
passwords
OTPs
cookies
API keys
wallet credentials
```

## Safe Example Policy

Use fake examples only.

Safe examples may include:

```text
https://example.com
0x1234567890abcdef
https://twitter.com/example
```

Unsafe examples:

```text
real seed phrase
real private key
real API token
real cookie
real wallet credential
```

## Stop Conditions

Stop review or publishing if a platform asks for:

```text
billing
card
upgrade
payout setup
secret token
private key
seed phrase
password
OTP
cookie
wallet connection
```
