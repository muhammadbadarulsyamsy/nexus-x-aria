# Publication Safety and Stop Rules v0.1

## Purpose

Protect NEXUS-X ARIA from unsafe publication behavior and platform/payment risk.

## Absolute Stop Conditions

Stop immediately if any platform asks for:

```text
billing
card
upgrade
payout setup
API token
secret
password
OTP
cookie
private key
seed phrase
wallet connection
```

## Prohibited Promotion

Do not use:

```text
spam
mass DM
fake claims
guaranteed scam detection
wallet protection claims
financial advice claims
aggressive outbound sales
misleading security claims
```

## Required Disclaimer

Use this or equivalent:

```text
NEXUS-X ARIA is heuristic-only. It does not guarantee safety, does not provide financial advice, and may produce false positives or false negatives.
```

## Safe Publication Behavior

```text
manual posting only
clear project identity
clear limitation
link to GitHub/demo/docs
no promises of revenue
no promises of safety
no request for sensitive credentials
```

## Safe Claim

```text
ARIA helps surface common airdrop risk signals from project metadata using explainable heuristic scoring.
```

## Unsafe Claim

```text
ARIA detects all scam airdrops and protects your wallet.
```
