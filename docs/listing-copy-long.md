# Listing Copy — Long

## Long Description

NEXUS-X ARIA is a transparent heuristic airdrop risk scoring API for Web3 projects.

It accepts project metadata such as project name, official URL, description, required tasks, blockchain chain, token contract, and social links. Based on these inputs, ARIA applies simple transparent heuristic rules to identify common risk signals.

The API returns a structured JSON response containing:

- risk score
- risk level
- verdict
- red flags
- safe actions
- summary

ARIA includes:

- a live landing page
- browser Try Page
- educational Examples page
- HTML documentation
- OpenAPI 3.0.1 specification
- internal test pack
- GitHub repository documentation

ARIA is designed for educational and informational use. It helps explain common airdrop risk signals such as suspicious domains, wallet connection requests, signing requests, seed phrase/private key requests, risky token approval language, promotional urgency, missing token contracts, missing social links, and unknown chains.

It also includes notes for false-positive handling, such as the zkSync domain substring case and educational token approval warnings.

## Important Limitation

NEXUS-X ARIA is not a complete security product. It is heuristic-only and can produce false positives and false negatives. It does not query blockchain explorers, domain reputation databases, wallet simulators, or external scam feeds.

It does not provide financial advice, investment advice, legal advice, or a guarantee of safety.

Users must always verify independently and must never share seed phrases, private keys, recovery phrases, passwords, OTPs, cookies, API keys, or wallet credentials.
