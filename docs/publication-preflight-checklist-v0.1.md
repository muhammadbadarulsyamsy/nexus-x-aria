# Publication Preflight Checklist v0.1

Use this before manually publishing the first ARIA post.

## Content Check

```text
[ ] headline is clear
[ ] ARIA is described as heuristic-only
[ ] no guaranteed safety claim
[ ] no financial advice claim
[ ] no wallet protection claim
[ ] no private keys requested
[ ] no seed phrases requested
[ ] no API keys requested
[ ] no false revenue promise
```

## Product Check

```text
[ ] mentions API
[ ] mentions metadata
[ ] mentions explainable scoring
[ ] mentions rule_hits or score_breakdown
[ ] mentions target audience
[ ] includes GitHub/repo link
[ ] includes try/demo link if available
```

## Platform Check

```text
[ ] platform is free
[ ] no billing/card required
[ ] no upgrade required
[ ] no payout setup required
[ ] no wallet connection required
[ ] no credential sharing required
```

## Publish Behavior

```text
[ ] manual post only
[ ] no automation
[ ] no mass DM
[ ] no spam
[ ] no repeated posting
[ ] no deceptive title
```

## After Publish

Record:

```text
date
platform
post URL
links used
initial reactions
comments/questions
any confusion
any safety concern
```

Use the publication review log after 7 and 14 days.
