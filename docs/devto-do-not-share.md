# Dev.to Do Not Share v0.1

Never share these in Dev.to, chat, GitHub issues, screenshots, or article drafts.

## Credentials

Do not share:

- passwords
- OTPs
- recovery codes
- cookies
- session tokens
- API keys
- GitHub tokens
- Cloudflare tokens
- database passwords
- private SSH keys

## Wallet Secrets

Do not share:

- seed phrase
- recovery phrase
- mnemonic phrase
- private key
- wallet JSON secret
- wallet password
- hardware wallet recovery words

## Billing or Identity Data

Do not share:

- credit card number
- billing address
- payout details
- tax ID
- identity document
- private account screenshot

## Local Sensitive Data

Avoid sharing:

- full local filesystem screenshots
- hidden config files
- `.env` files
- `hosts.yml`
- terminal logs containing tokens
- browser screenshots with account/session data

## Safe Rule

If unsure whether something is sensitive, do not share it.

Ask for a safer way to describe it without exposing the secret.
