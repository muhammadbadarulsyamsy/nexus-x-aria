# Platform Review Checklist v0.1

Use this before publishing or submitting NEXUS-X ARIA anywhere.

## Platform Basics

- [ ] Platform name recorded.
- [ ] Platform URL recorded.
- [ ] Terms of Service opened.
- [ ] Platform allows developer/API/open-source listings.
- [ ] Platform allows security education content.
- [ ] Platform allows Web3/crypto-related educational content.
- [ ] Platform does not require spammy promotion.

## Cost and Account Safety

- [ ] No billing required.
- [ ] No credit card required.
- [ ] No paid upgrade required.
- [ ] No payout setup required.
- [ ] No monetization required.
- [ ] No wallet connection required.
- [ ] No secret/API token required.
- [ ] No password/OTP/cookie requested outside normal login flow.
- [ ] No private key or seed phrase requested.

## Content Safety

- [ ] Listing says “heuristic”.
- [ ] Listing says “educational/informational”.
- [ ] Listing does not claim guaranteed safety.
- [ ] Listing does not claim detecting all scams.
- [ ] Listing does not claim financial advice.
- [ ] Listing includes disclaimer.
- [ ] Listing includes limitations if space allows.
- [ ] Listing does not encourage connecting real wallets.
- [ ] Listing does not encourage speculation/trading.

## Link Safety

- [ ] GitHub link correct.
- [ ] Live API link correct.
- [ ] Try Page link correct.
- [ ] Examples link correct.
- [ ] OpenAPI link correct.
- [ ] Test Pack link correct.
- [ ] No local file path included.
- [ ] No credential included.
