## Stop Conditions

Stop if any platform, service, or workflow asks for:

```text
billing
credit card
paid upgrade
payout setup
API token
secret
private key
seed phrase
password
OTP
cookie
wallet connection
```

## Important Rule

NEXUS-X ARIA must be described as:

```text
a transparent heuristic educational MVP
```

Do not describe it as:

```text
a guaranteed scam detector
a wallet protection guarantee
financial advice
investment advice
a certified security auditor
an enterprise security product
```
