# Disclaimer

NEXUS-X ARIA is provided for informational, educational, and experimental purposes only.

## Not Financial or Investment Advice

The output of this API is not financial advice, investment advice, trading advice, legal advice, or a recommendation to participate in any airdrop, token sale, protocol, or Web3 activity.

Do not use ARIA as the sole basis for making financial or security decisions.

## No Guarantee of Safety

ARIA uses transparent heuristic scoring rules. It may produce false positives or false negatives.

A low risk score does not guarantee that an airdrop is safe.

A high risk score does not prove malicious intent.

Users must perform their own research and verify information through official and trusted sources.

## No Wallet or Fund Management

ARIA does not:

- connect to wallets
- sign transactions
- trade assets
- mine cryptocurrency
- hold funds
- manage private keys
- request seed phrases
- request recovery phrases
- request passwords
- request OTPs
- request cookies or login tokens

## User Responsibility

You are responsible for protecting your own wallet, credentials, devices, and funds.

Never share:

- private keys
- seed phrases
- recovery phrases
- passwords
- OTPs
- login cookies
- API keys
- exchange credentials

## MVP Status

NEXUS-X ARIA v0.1.5 is an MVP. It is not a complete security product and should be treated as an early-stage risk analysis tool.
